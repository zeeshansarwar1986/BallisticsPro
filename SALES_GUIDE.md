## 📄 فروخت کا طریقہ / Sales Process

### مرحلہ 1: ابتدائی رابطہ (Initial Contact)
- سرکاری لیٹر یا ای میل کے ذریعے رابطہ کریں
- NDA (Non-Disclosure Agreement) پر دستخط کروائیں
- ایک مفت Demo Session دیں (1 گھنٹہ)

### مرحلہ 2: تجویز (Proposal)
- تنظیم کی ضروریات کا جائزہ لیں
- کسٹم لائسنس پیکج تجویز کریں
- سرکاری Quotation بھیجیں

### مرحلہ 3: معاہدہ (Agreement)
- License Agreement تیار کریں
- End-User Certificate حاصل کریں (سیکورٹی ادارے کے لیے لازمی)
- Payment Terms طے کریں (50% Advance، 50% Delivery)

### مرحلہ 4: فراہمی (Delivery)
- APK فائل ڈیوائسز پر انسٹال کریں
- تربیت فراہم کریں (Training Session)
- Technical Support معاہدہ شامل کریں

## 🚀 Android APK تیار کرنے کا طریقہ / How to Build Android APK

چونکہ ایپ PWA ہے، آپ اسے اصل APK میں تبدیل کرنے کے دو طریقے ہیں:

### طریقہ 1: Capacitor (پیشہ ور طریقہ)
```bash
# Node.js انسٹال کریں: https://nodejs.org
npm install -g @capacitor/cli
npm install @capacitor/core @capacitor/android
npx cap init BallisticsPro com.ballistics.pro
npx cap add android
npx cap sync
npx cap open android
# Android Studio میں Build > Generate Signed APK
```

### طریقہ 2: آن لائن Web to APK (فوری طریقہ)
1. `D:\Task\sniper cam\` فولڈر کو zip کریں
2. ویب سائٹ پر اپ لوڈ کریں: https://gonative.io یا https://appsgeyser.com
3. APK ڈاؤن لوڈ کریں
4. ڈیوائس پر انسٹال کریں

> **نوٹ:** PWA کو Chrome پر کھولیں اور "Add to Home Screen" کریں — یہ APK جیسا کام کرتا ہے!
